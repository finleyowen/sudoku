# C++ sudoku solver

**Under active development**

A sudoku solver written in C++ using graphing algorithms on an implicit unweighted graph.

## Building from source

1. Clone the repository:
	```
	git clone http://github.com/finleyowen/sudoku
	```

2. The Makefile is setup for building with `g++`. Either ensure it is installed or edit the Makefile to use your desired C++ compiler.

3. Build and run!
	```
	make # build
	./sudoku # run
	```
